# 🚀 Lesson 01: C++ Introduction & Setup

Welcome to the first lesson of the C++ learning series. This module covers the core fundamentals of C++, its significance in modern computer science, and the step-by-step process to set up your development environment.

---

## 🔍 What is C++?

**C++** is a high-level, general-purpose, object-oriented programming language developed by **Bjarne Stroustrup** at Bell Labs in 1979 as an extension of the C programming language.

### Key Characteristics:

- **Cross-Platform:** Code written in C++ can be compiled and executed on various operating systems like Windows, macOS, and Linux.
- **Multi-Paradigm:** It supports procedural, object-oriented (OOP), and generic programming.
- **Middle-Level Language:** It bridges the gap between low-level languages (direct hardware manipulation) and high-level languages (user-friendly syntax).

---

## ⚡ Why Use C++?

C++ is highly favored in computer science and software engineering due to its performance and versatility.

| Reason                  | Description                                                                                                                                        |
| :---------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Speed & Performance** | Being a compiled language, it executes instructions exceptionally fast, making it ideal for performance-critical systems.                          |
| **Memory Control**      | Provides direct access to hardware and manual memory management (via pointers), giving developers total control over system resources.             |
| **Wide Industry Use**   | It is the backbone behind operating systems (Windows, Linux), game engines (Unreal Engine), web browsers, and high-frequency trading applications. |
| **Rich Ecosystem**      | Massive standard template library (STL) and active global community support.                                                                       |

---

## 🛠️ How to Install & Set Up C++

To write and execute C++ programs, your system requires two main components:

1. **A Compiler:** Translates your C++ source code into machine-readable machine language (e.g., GCC/MinGW, Clang, MSVC).
2. **An IDE (Integrated Development Environment) / Text Editor:** The environment where you write your code.

### 1️⃣ C++ Compiler Installation

#### For Windows (MinGW):

1. Download the **MinGW-w64** installer via MSYS2 or directly from SourceForge.
2. Install the C++ compiler packages (`gcc`, `g++`, `gdb`).
3. Add the compiler's binary directory path (e.g., `C:\msys64\mingw64\bin`) to your system's **Environment Variables (Path)**.
4. Verify the installation by running the following command in your Terminal/Command Prompt:
   ```bash
   g++ --version
   ```

## For macOS:

Open the terminal and install Xcode Command Line Tools using:

```Bash
    xcode-select --install
```

## For Linux (Ubuntu/Debian):

Update the package list and install the build-essential package:

```Bash
sudo apt update
sudo apt install build-essential
```
